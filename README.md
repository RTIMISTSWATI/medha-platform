# Medha

From first problem to dream career — practice coding, prepare for exams and interviews, build real-world skills, and get hired with confidence.

## About

Medha is an AI-powered developer growth platform designed for students, coders, and professionals. It combines coding practice, skill analysis, resume intelligence, interview preparation, competitive exam preparation, and job opportunities into one complete ecosystem.

## Core Features

* Coding practice platform
* Live contests and leaderboards
* AI skill analysis dashboard
* Resume parser and ATS score checker
* Interview preparation system
* Competitive exam preparation
* Job and internship portal
* Gamification with XP, streaks, and badges

## Tech Stack

* Frontend: React + Vite + Tailwind CSS
* Backend: Node.js + Express.js
* Database: MongoDB
* AI Engine: Python + FastAPI
* Deployment: Vercel + Render
