## 🧠 Overview

**Medha** is a unified platform designed to help students and developers stay consistent, practice effectively, and work in a distraction-free environment.

It combines:
- coding practice  
- task planning  
- progress tracking  
- AI-assisted learning  

— all inside a **cozy, dark-academia inspired interface**.

---

## 🚀 Features

### 💻 Coding Playground
- Multi-problem coding interface (DSA focused)
- Clean editor with focus mode
- Output panel with run feedback (status + animations)
- AI-powered code assistant *(Explain / Improve / Debug — in progress)*

---

### 📊 Dashboard
- Study hours tracking  
- Task completion stats  
- Productivity insights  
- Weekly consistency view  

---

### 📝 Planner
- Daily task management  
- Streak tracking  
- Motivational system (dynamic quotes)  

---

### 🎧 Ambient Experience
- Spotify integration for focus sessions  
- Minimal, distraction-free UI  
- Designed for long, deep work sessions  

---

## 🎨 Design Philosophy

Medha is built around:

- **Focus-first UX** → remove distractions, highlight what matters  
- **Dark academia aesthetic** → warm, calm, immersive  
- **Glassmorphism layers** → depth without clutter  
- **Consistency over complexity**
